# AI-Based Resume Screening System (SAP ABAP Cloud)

An intelligent recruitment solution developed using SAP ABAP Cloud and Artificial Intelligence to automate resume screening, candidate analysis, skill extraction, and profile matching processes.

---

## 📌 Project Overview

This project focuses on simplifying and improving the recruitment process by automating resume analysis using AI and SAP ABAP Cloud technologies. The system extracts candidate information, identifies relevant skills, and matches profiles based on job requirements.

The application reduces manual screening effort and improves hiring efficiency through intelligent resume processing and automated candidate evaluation.
